# Higher Signal

Higher Signal is a Streamlit application that helps revenue teams prioritize target accounts from CRM exports.

## Features

- Upload target account CSVs
- Match prospects to similar customers
- Generate account prioritization signals
- Surface GTM insights from account data

## Built With

- Python
- Streamlit
- Pandas
- Scikit-Learn

## Why I Built It

As an enterprise AE, I wanted a faster way to turn raw territory data into actionable account strategy using AI-assisted development tools.
